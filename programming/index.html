Here is a list of programming languages that I personally like:

  • Standard ML, OCaml: Easy to learn functional programming languages. If you are new to programming I recommend starting with one of these.
  • C: Low-level programming language in which many foundational software is written.
  • Python: The scripting language. Python provides libraries to do almost everything under the sun.
  • Common Lisp: The ultimate interactive program construction experience. Common Lisp development environments allow you to interact with your partially constructed program to figure out the missing pieces. The language is also extermely malleable and allows you to easily extend the language with your own features.
